# Changelog

## [1.0.0] - 2026-01-01

### Added
- Initial release
- Window class with title, icon, geometry, state management
- HeaderBar with title, subtitle, icon, window control buttons
- WindowEffect with blur, mica, acrylic, corner radius, dark mode
- WindowHelper utilities for screen and platform operations
- QML module (Lingmo.Window) with Window, HeaderBar, Dialog, WindowEffect
- Cross-platform support (Windows, X11, Wayland)
- Platform-specific window effects (DWM on Windows)
- CMake build system with find_package integration
- Debian and RPM packaging
- GitHub Actions CI (build, test, package, ABI)
